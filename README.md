# Fit4Finance
  
[fit4finance.io](https://www.fit4finance.io)
 
**Gewinner „KARDEA!" 2026** — HTL Anichstraße, 3. Klasse Wirtschaftsabteilung
 
---
 
Fit4Finance ist ein browserbasiertes Lernspiel, in dem Jugendliche den Aktienhandel anhand realer historischer Marktszenarien (z. B. Corona-Crash, Dotcom-Blase) nachspielen können. Während des Spiels werden historische Nachrichten und Ereignisse in Echtzeit eingeblendet. Gehandelt wird ausschließlich mit virtuellem Geld. Ergänzend steht ein integriertes Wiki mit grundlegenden Börsenbegriffen zur Verfügung.
  
React · Java (Javalin REST) · SQLite · Yahoo Finance API · Docker
